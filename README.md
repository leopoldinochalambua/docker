# Introdução 

Olá pessoal, bem vindo aqui!.

Esse é um material que crio com base no curso de docker, ministrado no YouTube pelo professor `Caio Delgado`. O curso é feito em virtual box, e eu estou a usar o `qemu-kvm` como o meu privider. Por isso decidi criar esse conteúdo e mostrar todo passo a passo.

Durante os primeiros dias, vou me aprofundar em conhecer as bases do Docker através da compreensão da arquitetura básica e de como configurar um ambiente Docker. ser capaz de criar e gerir imagens,  containers Docker e trabalhar com o Docker Hub e o Docker Cloud, onde vou partilhar também as imagens que vou criar.

Este capítulo também abordará instalações, onde vou me focar na instalação baseada em debian, e usar o qemu-kvm, para garantir que meu ambiente esteja configurado e pronto para o desenvolvimento do Docker.

Vou mostrar como instalei o `vagrant`, o `virt-manager` _deu uma certa dor de cabeça, mais no fim terminou bem_, a adição das imagesn e a instalação do git. Abaixo pode ver o programa do que vai ser abordado. Vou procurar subir todos os capítulos no git, assim que estiverem terminados. 

> **OBS**: Além desse tema (docker), tenhos outros treinamentos em andamento, o que pode me impedir de atualisar os repositóorios com uma certa regularidade.
